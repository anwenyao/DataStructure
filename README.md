# Sorting(排序)
### 常见的八种排序:
**插入排序：**
- 直接插入排序
- 希尔排序

基本思想：先将整个待排序记录序列分割成若干子序列分别进行直接插入排序，待整个序列中的记录“基本有序”时，再对全体记录进行一下直接插入排序。

**选择排序：**
- 简单选择排序
- 堆排序

**交换排序：**
- 冒泡排序
- 快速排序

基本思想：通过一趟排序将待排记录分割成独立的两部分，其中一部分记录的关键字均比另一部分记录的关键字小，则可分别对着两部分记录继续进行排序，以达到整个序列有序。
